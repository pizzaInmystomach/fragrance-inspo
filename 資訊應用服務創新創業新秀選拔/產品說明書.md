# 產品說明書

## 一、公司/團隊介紹

香氛靈感團隊致力於運用人工智慧技術，為消費者提供個人化的香水推薦服務。我們結合深度學習、自然語言處理與香氛美學，開發出全球首創的角色型香水匹配系統。團隊成員具備資訊工程、人工智慧、香氛調香及市場行銷等跨領域專業背景，致力於打造創新的嗅覺體驗平台，讓每個人都能找到最適合自己個性與風格的專屬香氛。

## 二、創新與創意

### 2.1 獨特創新概念

香氛靈感系統開創性地將心理學角色分析與 AI 技術結合，透過用戶輸入的虛擬或真實角色（如動漫人物、電影角色、歷史人物等），分析其性格特質、風格偏好和氣質特徵，進而推薦最符合該角色特質的香水產品。

### 2.2 核心競爭優勢

**技術創新**：
- 首創角色驅動的香水推薦演算法  
- 運用大型語言模型進行深度角色心理分析  
- 建立香氛特質與人格特質的智能匹配機制  

**使用者體驗創新**：
- 將抽象的香氛選擇具象化為角色投射  
- 提供沉浸式的香氛探索體驗  
- 個人化的香氛故事敘述功能  

**市場差異化**：
- 跳脫傳統香調分類的推薦模式  
- 結合娛樂性與實用性的消費體驗  
- 開創香水產業的新興消費場景  

## 三、產品的功能與效果

### 3.1 核心功能

**角色分析引擎**：
- 智能解析角色的性格特質、風格特點及氣質特徵  
- 支援多元角色類型：動漫、電影、文學、歷史人物等  
- 提供詳細的角色心理分析報告  

**香水匹配系統**：
- 基於角色特質的智能香水推薦  
- 涵蓋前中後調的完整香氛分析  
- 個性化的推薦理由說明  

**香氛描述生成**：
- AI 生成專屬的香氛體驗描述  
- 融合角色特質的香氛故事創作  
- 豐富的感官體驗文字敘述  

### 3.2 預期效果

**對消費者**：
- 降低香水選購的試錯成本  
- 提升香氛選擇的滿意度  
- 增加香水消費的趣味性和個人意義  

**對香水品牌**：
- 提供新穎的行銷推廣工具  
- 增進消費者對產品的理解和認同  
- 創造差異化的品牌體驗  

**對零售通路**：
- 降低銷售人員的專業門檻  
- 提升銷售轉換率  
- 創造創新的零售體驗  

## 四、技術可行性

### 4.1 技術架構

**AI 核心技術**：
- 採用 LangChain 框架整合大型語言模型  
- 運用 Ollama 本地部署確保資料隱私  
- 實現模組化的 AI 處理鏈設計  

**後端技術**：
- FastAPI 高效能 Web 框架  
- 非同步處理提升系統響應速度  
- RESTful API 設計確保擴展性  

**資料處理**：
- 結構化香水資料庫設計
